This is a version of TinyXMl compiled in Visual Studio 2010.

Unzip and copy contents into your project's folder, so that '.lib' and '.h' files are correctly placed in the 'lib' and 'include' folders (check)

If for some reason this version is not compatible with your project, 
download the source code from the link below, compile the library, 
and copy the resulting TinyXML.lib, tinystr.h and tinyxml.h to the 
corresponding 'lib' and 'include' folders of your project.

Source TinyXML: http://www.sourceforge.net/projects/tinyxml
